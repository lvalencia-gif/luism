# Luis M's Guessing Game

A fun number guessing game written in Python, featuring Roman numeral displays!

## Features

- Guess a secret number between 1 and 20
- Get feedback on whether your guess is too high or too low
- See the secret number revealed in Roman numerals when you win
- Roll a virtual die at the end, with the result shown in Roman numerals
- Robust input validation to handle invalid entries

## How to Play

1. Run the game: `python guess_game.py` or `python "import random.py"`
2. Enter your guesses when prompted
3. Keep guessing until you find the secret number!
4. Enjoy the dice roll at the end

## Files

- `guess_game.py`: Main game with basic input handling
- `import random.py`: Enhanced version with better error handling
